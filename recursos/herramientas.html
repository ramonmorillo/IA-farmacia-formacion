🤖 Asistentes de IA Generativa

Herramientas de IA conversacional que pueden ayudar con tareas administrativas, búsqueda de información y generación de borradores. Ninguna está validada para uso clínico directo.

Características a buscar en un asistente de IA:

  • ✅ Actualización frecuente: Modelos actualizados regularmente
  • ✅ Capacidad de razonamiento: No solo generación de texto
  • ✅ Transparencia: Indica cuando no sabe algo
  • ✅ Privacidad: Política clara sobre datos introducidos
  • ✅ Acceso a web (opcional): Puede buscar información actualizada

Tipos de asistentes disponibles (enero 2025):

Tipo Ejemplos Ventajas Limitaciones
Asistentes generales ChatGPT, Claude, Gemini, Copilot Amplio conocimiento general, fácil acceso No especializados en medicina
Con acceso a web Perplexity, Copilot, algunos con plugins Información actualizada, citan fuentes Calidad depende de las fuentes web
Especializados médicos Algunos en desarrollo clínico Entrenados con literatura médica Aún no validados para decisiones
⚠️

Criterios para elegir:

  • Para aprendizaje personal: Cualquier asistente general de calidad
  • Para trabajo con datos sensibles: Verifica política de privacidad, evita introducir datos de pacientes
  • Para investigación: Prefiere los que citan fuentes verificables
  • Para farmacia hospitalaria: Consulta con tu departamento de IT sobre herramientas aprobadas

📚 Bases de Datos Farmacéuticas Oficiales (para VERIFICAR IA)

Estas son tus fuentes de VERDAD. Úsalas SIEMPRE para verificar información de IA.

🥇 Nivel 1: Fuentes primarias oficiales

España:

  • CIMA (AEMPS): https://cima.aemps.es
    Fichas técnicas oficiales, prospectos, información de medicamentos autorizados en España
  • Bot PLUS: Base de datos de medicamentos comercializados
    Información de presentaciones, precios, financiación

Internacional:

  • EMA (Agencia Europea): https://www.ema.europa.eu
    Autorizaciones europeas, informes de evaluación
  • FDA (EE.UU.): https://www.fda.gov
    Etiquetados, alertas de seguridad

🥈 Nivel 2: Bases de datos clínicas validadas

  • Micromedex (IBM): Interacciones, dosificación, toxicología
    Acceso vía suscripción hospitalaria o universitaria
  • UpToDate: Información clínica basada en evidencia
    Revisiones de expertos actualizadas regularmente
  • Lexicomp: Información de medicamentos, interacciones
    Usado en muchos hospitales
  • DynaMed: Herramienta de soporte a la decisión clínica
    Síntesis de evidencia científica

📖 Literatura científica:

  • PubMed: https://pubmed.ncbi.nlm.nih.gov
    Base de datos de literatura biomédica (gratuito)
  • Cochrane Library: https://www.cochranelibrary.com
    Revisiones sistemáticas de alta calidad
  • ClinicalTrials.gov: https://clinicaltrials.gov
    Registro de ensayos clínicos

🧮 Calculadoras Clínicas Validadas

Usa ESTAS para cálculos, NO IA generativa.

Calculadoras generales:

  • MDCalc: https://www.mdcalc.com
    Scores clínicos, calculadoras de dosis, riesgo cardiovascular
  • QxMD Calculate: App móvil con calculadoras médicas validadas
    Función renal, ajustes de dosis, scores
  • GlobalRPh: https://www.globalrph.com
    Calculadoras farmacéuticas, conversiones, diluciones

Específicas:

  • Aclaramiento de creatinina: Cockcroft-Gault, CKD-EPI (en MDCalc)
  • Ajuste de dosis renal: Calculadoras específicas por fármaco en Micromedex
  • Nutrición parenteral: Calculadoras hospitales de referencia
  • Farmacocinética: Software especializado (DoseMe, MwPharm, etc.)
⚠️

Nunca uses IA generativa para:

  • Cálculos de dosis pediátricas
  • Cálculos de quimioterapia
  • Ajustes complejos de anticoagulantes
  • Cálculos que afectan decisiones inmediatas

Usa calculadoras validadas + verificación con colega.

📋 Guías Clínicas Oficiales

Fuentes autorizadas para verificar recomendaciones terapéuticas.

Internacionales:

  • NICE (Reino Unido): https://www.nice.org.uk
    Guías clínicas basadas en evidencia
  • ADA (American Diabetes Association): Guías de diabetes
    Actualizadas anualmente
  • ESC (European Society of Cardiology): Guías cardiología
    Referencia europea en enfermedades cardiovasculares
  • IDSA (Infectious Diseases Society): Guías de infecciosas
    Tratamientos antimicrobianos

Españolas (Sociedades Científicas):

  • SEMICYUC: Cuidados intensivos
  • SEPAR: Neumología
  • SEN: Neurología
  • GEMA: Asma (guía española)
  • GESIDA: VIH/SIDA

📊 Herramientas de Gestión y Documentación

Para registro de uso de IA:

  • Excel/Google Sheets: Plantillas para tracking de consultas
    Descarga nuestras plantillas en la sección de documentación
  • Notion/Obsidian: Bases de conocimiento personal
    Para organizar prompts reutilizables y aprendizajes
  • Sistemas hospitalarios: Integración con tu software de gestión farmacéutica
    Consulta con IT de tu hospital

Gestores bibliográficos:

  • Zotero: Gratuito, código abierto
    Ideal para organizar referencias de estudios
  • Mendeley: Gratuito con funciones sociales
    Buena integración con Word
  • EndNote: Profesional (de pago)
    Usado en entornos académicos

🔌 Extensiones de Navegador Útiles

  • Unpaywall: Acceso a artículos científicos en abierto
    Identifica versiones gratuitas legales de papers
  • JANE (Journal/Author Name Estimator): Sugerencias de revistas para publicar
    Basado en tu abstract
  • Grammarly/LanguageTool: Corrección de textos en inglés/español
    Útil al redactar documentos científicos

Cursos introductorios gratuitos:

    Introducción no técnica a IA (disponible en español)

  • Webinars de hospitales de referencia
  • Literatura emergente en revistas farmacéuticas

💡

Recomendación final:

Crea tu propio "kit de herramientas" personalizado con 3-5 recursos que uses regularmente: 1-2 asistentes de IA para borradores, 2-3 bases de datos para verificar, 1 calculadora de confianza. Conócelos bien y úsalos de forma consistente.

⚠️ Las herramientas de IA son auxiliares. Siempre verifica con fuentes oficiales y usa calculadoras validadas.
