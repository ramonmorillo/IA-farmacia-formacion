# Changelog

## 3.0 - 2026-07-14

- Rediseño de la aplicación como entorno formativo modular para farmacia hospitalaria.
- Nueva arquitectura de datos centralizados y componentes reutilizables.
- Tres niveles acumulativos y ruta avanzada de farmacia hospitalaria.
- Nuevos módulos: selección de herramientas, contexto, evidencia, multimodalidad, agentes, seguridad, evaluación y gobernanza.
- Casos prácticos rediseñados con errores deliberados y rúbricas.
- Progreso local, favoritos, búsqueda, filtros, modo oscuro, presentación, modo docente e informe descargable.
- Plantillas descargables de gobernanza institucional.
- Páginas puente para conservar URL existentes.
