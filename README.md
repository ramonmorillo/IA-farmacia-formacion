# IA en Farmacia - Guía de Uso Responsable

**Creado por Ramón Morillo | Enero 2026**

Herramienta formativa web para talleres de inteligencia artificial dirigidos a profesionales farmacéuticos.

## 📋 Descripción

Esta herramienta educativa tiene como objetivo formar a farmacéuticos (estudiantes, comunitaria, atención primaria y hospitalaria) en el uso responsable, crítico y seguro de la IA en el trabajo farmacéutico.

### Principios fundamentales:
- ✅ **NO es una herramienta clínica**
- ✅ **NO debe apoyar decisiones terapéuticas**
- ✅ **NO debe usar datos reales de pacientes**
- ✅ **Todo el contenido es formativo con ejemplos ficticios**

## 🎯 Público objetivo

- Estudiantes de Farmacia
- Farmacéuticos de oficina de farmacia/comunitaria
- Farmacéuticos de atención primaria
- Farmacéuticos hospitalarios

## 🚀 Características

### Contenido modular
- **M1 - Fundamentos**: Conceptos básicos, capacidades/límites, marco ético
- **M2 - Casos de uso**: Permitidos, prohibidos y zona gris
- **M3 - Prácticas seguras**: Prompting, verificación, documentación
- **M4 - Aplicaciones**: Información, automatización, formación, investigación

### Rutas personalizadas
Cada perfil tiene una ruta formativa adaptada a su contexto profesional

### Casos interactivos
5 casos prácticos con feedback inmediato para aprender haciendo

### Sistema semafórico
- 🟢 Verde: Casos de uso permitidos
- 🟡 Amarillo: Requiere supervisión
- 🔴 Rojo: Casos prohibidos

## 🛠️ Tecnología

- **HTML5 + CSS3**: Estructura y diseño
- **JavaScript vanilla**: Interactividad (sin frameworks)
- **GitHub Pages**: Hosting gratuito
- **Sin backend**: Todo funciona en el cliente
- **Sin dependencias externas**: Funciona offline

## 🚀 Cómo usar

### Opción 1: Ver en local
1. Descarga el proyecto
2. Abre `index.html` en tu navegador
3. ¡Listo! Funciona sin servidor

### Opción 2: Publicar en GitHub Pages
1. Crea un repositorio en GitHub
2. Sube todos los archivos
3. Ve a Settings → Pages
4. Selecciona la rama `main` y carpeta `/ (root)`
5. Guarda y espera 1-2 minutos
6. Tu web estará en `https://tu-usuario.github.io/nombre-repositorio`

## ⚖️ Límites éticos incorporados

### Diseño
- Banners de advertencia en todas las páginas
- Código de colores semafórico
- Checkpoints de responsabilidad

### Contenido
- Lenguaje normativo claro ("NO se debe", no "no se recomienda")
- Ejemplos siempre ficticios y etiquetados
- Referencias normativas explícitas (RGPD, deontología)

### Arquitectura
- Sin almacenamiento de datos externos
- Sin integración con APIs de IA reales
- Sin capacidad de procesar datos sensibles

## 🔒 Privacidad y seguridad

- ✅ No recopila datos de usuarios
- ✅ No se conecta a servicios externos
- ✅ No requiere registro ni login
- ✅ Todo funciona en el navegador (cliente)
- ✅ Cumple RGPD por diseño
